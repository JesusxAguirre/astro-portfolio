## Overview
**IPTV App** is a sleek, modern dark-themed Flutter application designed for effortless movie streaming. Inspired by popular platforms like Netflix, it offers a user-friendly interface for discovering and watching movies.

Users faced the following challenges:
1. Difficulty in quickly finding popular movies or TV series with an intuitive UI and UX.
2. Uncertainty in choosing what to watch during limited free time.

## üéØ Goal
1. Develop a simple and intuitive mobile application that can be easily used by users with minimal training.
2. Integrate the Movie DB API to provide up-to-date movie information.
3. Implement offline functionality, allowing users to browse previously loaded content without an internet connection.

---

## ‚ú® Solution

### Understanding User Needs
Through user feedback, we identified key requirements:
- **Intuitive Navigation:** Easy access to movie categories and details.
- **Responsive Design:** Seamless experience across various devices and screen sizes.

### Core Features
The app includes:
- **Movie Categories:** Browse movies by genres such as Trending, Top Rated, and Now Playing.
- **Search Functionality:** Quickly find movies by title or actor.
- **Movie Details:** Access comprehensive information including ratings, cast, and reviews.

### Design and Development Process
The app was developed using:
- **Flutter:** For building natively compiled applications for mobile.
- **Riverpod:** For state management, ensuring a scalable and testable architecture.
- **GoRouter:** For declarative routing, simplifying navigation.
- **Movie DB API:** For fetching real-time movie data.
- **Shimmer:** For loading animations, enhancing user experience during data fetching.

---

## üß™ User Testing
We tested the app prototype with 10 participants and gathered insights:
- **Ease of Navigation:** 9/10 participants found the app easy to navigate.
- **Search Functionality:** Users appreciated the quick and responsive search feature.
